# 🛒 Zepto SQL Data Analysis Project

## 📌 Overview

This project focuses on analyzing a real-world Zepto dataset using SQL (PostgreSQL).
The goal is to clean the data, perform transformations, and extract meaningful business insights.

---

## 🧰 Tools & Technologies

* PostgreSQL (pgAdmin)
* SQL
* CSV Dataset

---

## 📊 Key Analysis Performed

* Data cleaning and handling NULL values
* Product category analysis
* Price and discount analysis
* Stock availability insights
* Identifying high-demand products

---

## 📂 Dataset Features

The dataset includes:

* Product category
* Product name
* MRP (Maximum Retail Price)
* Discount percentage
* Available quantity
* Discounted selling price
* Weight (in grams)
* Stock availability

---

## 🔍 Sample Queries

* Find distinct product categories
* Analyze products with highest discounts
* Check out-of-stock items
* Identify NULL values in dataset

---

## 📈 Insights

* Certain categories have higher discounts than others
* Stock availability varies significantly across products
* Pricing strategies differ by category

---

## 🚀 How to Use

1. Import the dataset into PostgreSQL
2. Run the SQL queries provided
3. Analyze outputs and insights
